Personal website of **Dilanka Kaushal Hewage (r3vpwnx)** - Red Team Operator, Security Researcher, CTF Player

This repository contains the source code for my personal website showcasing my work in offensive security, red team operations, and cybersecurity research.

I'm a Red Team Operator at Dubai Health Authority, specializing in:

- Active Directory Security
- CTF Competitions

## Website Features

- **Catppuccin Theme** - Clean, whitish background with blue/purple accents
- **Security-Focused Content** - Blog posts, research, and red team projects
- **Certifications Showcase** - OSEE, OSED, CRTL, CRTO, OSEP, OSCP, and more
- **Responsive Design** - Mobile-friendly and accessible

## Technology Stack

- **Static Site Generator:** Jekyll
- **Theme:** Customized Minimal Mistakes theme
- **Hosting:** GitHub Pages
- **CSS:** LESS preprocessor with custom Catppuccin colors

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# Access at http://localhost:4000
```
